Это большой тестовый документ для проверки прогресс-бара. Он содержит несколько предложений и абзацев. Система должна показать прогресс загрузки файла, а затем этапы обработки. После загрузки файла на сервер происходит декодирование контента, разбиение на чанки, обработка каждого чанка с эмбеддингами, сохранение в базу данных. Каждый этап должен быть виден пользователю.

Второй абзац содержит дополнительную информацию о тестировании. Здесь мы проверяем работу всех компонентов системы: фронтенда с прогресс-баром, сервера с логированием, базы данных Neo4j. Все должно работать слаженно.

Третий абзац нужен для создания достаточно большого документа. В реальных условиях документы могут быть намного больше, содержать сотни страниц текста. Система должна эффективно обрабатывать такие объемы, показывая пользователю понятный прогресс операции.
