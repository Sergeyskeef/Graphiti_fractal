Новый уникальный тестовый документ для проверки тайминга системы. Этот файл должен показать реальное время обработки чанков. Система измеряет время от начала до конца каждого этапа: чанкирование, эмбеддинги, Graphiti операции.

Документ содержит достаточно текста для создания нескольких чанков. Каждый чанк будет обрабатываться отдельно с измерением времени. Ожидается, что основное время уйдет на Graphiti.add_episode операцию.

Финальный отчет покажет:
- Общее время обработки (WALL)
- Время ingest операции (INGEST) 
- Количество чанков (CHUNKS)
- Среднее время на чанк (AVG_CHUNK)
- Максимальное время на чанк (MAX_CHUNK)
- Детальное время по каждому чанку (PER_CHUNK)
